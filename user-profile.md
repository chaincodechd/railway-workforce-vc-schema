# UserProfile Schema

This document describes the **UserProfile** schema, including personal details such as full name, email, mobile number, and photo URL.

---

## full_name

`full_name` is the complete name of the user.

```json
"full_name": "Jane Doe"
```

## email

`email` is the user's email address.

```json
"email": "jane.doe@example.com"
```

## mobile_number

`mobile_num
ber` is the user's mobile phone number (including country code if applicable).

```json
"mobile_number": "+91-9876543210"
```

## photo_url

`photo_url` is a URL pointing to the user's profile photograph.

```json
"photo_url": "https://example.com/photos/jane.jpg"
```
